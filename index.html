What is OpenSA?
OpenSA is a mod manager for GTA San Andreas. This is a similar application to OpenIV. At this moment is in very early stage so is avillable only mod installer. You can create own mod installer by checking documentation for help. Also this is not a unsport try to copy the original idea for OpenIV, this app looks similar but only looks like that.
